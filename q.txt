1. Установите пакет в frontend/ (в секцию devDependencies): npm install -D openapi-typescript

2. Добавьте скрипт автоматизации в frontend/package.json:

json"scripts": {
  "dev": "vite",
  "build": "tsc && vite build",
  "generate-types": "openapi-typescript http://127.0.0 -o src/types/api.ts"
}

Шаг 3. Запуск и сквозной результат
Теперь процесс работы между вами и фронтенд-разработчиком выглядит так:
Вы запускаете локальный сервер бэкенда (python manage.py runserver).
Фронтендер в своей консоли запускает одну команду:
bash

npm run generate-types

Утилита делает быстрый HTTP-запрос к вашему бэкенду, анализирует структуру DRF и автоматически создает файл src/types/api.ts,
в котором будут абсолютно все модели, сериализаторы
