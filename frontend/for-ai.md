Проект: Система учета закупок в кампании

Стек:

- React
- shadcn/ui
- zod
- rhf
- tanstack/react-query

Структура:

- app/
- features/
- shared/

Правила:

- комментарии на русском
- короткие функции
- не использовать глобальные переменные

Текущий статус:

- Создана главная страница закупок
- Кнопка для добавления новой записи в таблицу закупок
- Форма добавления разбита на разные блоки (основная информация, договоры, планирование) с табами для перехода по блокам
- Секция договоров имеет поля и таблицу, которая по кнопке добавляется с input полями из tanstack-table, там находятся поля планирование (сколько есть денег),оплата (сколько оплатили), колонки -это кварталы с разбитием суммы на кварталы и итого. Может содержать 30 договоров за раз, так что создал таблицу для договоров
